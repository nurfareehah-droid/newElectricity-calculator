# newElectricity-calculator


This project is a PHP-based Electricity Calculator that calculates electricity usage and charges based on user input.

Features:
- Calculate Power (kW)
- Calculate Energy (kWh)
- Calculate Electricity Charges
- Hourly calculation (1–24 hours)
- Daily summary
- Uses functions in PHP

Input:
- Voltage (V)
- Current (A)
- Current Rate (sen/kWh)

Technology Used:
- PHP
- HTML
- Bootstrap 5
